VERBS

only a few examples of specific lexemes so far

mothers: root

patterns
  (no) (me, te, ...) golpean, golparon, ...
  (no) (me, te, ...) (estoy, ...) golpeando
  (no) (me, te, ...) (he, has, ...) golpeado
  (no) me siento, ...
  limitation: only one pronoun object

features:
  subject person-number
  object person-number
    -1 for intransitive
    (0,0,0)/(0,0,1) [3s/p] for transitive with no pronoun object
  negation
  tense/aspect

agreement
  subject-reflexive object agreement
    e.g., te sentaste

-> Gn
translation agreement
  tense/aspect
    preterite -> -'akue
    present -> 0
    progressive -> -'aína
    perfect -> -ma
  subject
  object (3p objects translate as chupe(kuéra) or 0)
  negation
